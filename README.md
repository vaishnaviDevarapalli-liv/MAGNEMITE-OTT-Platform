🎬 MAGNEMITE – Premium OTT Streaming PlatformMAGNEMITE is a modern, high-performance web application designed to simulate a premium movie streaming experience. Built with a "cinematic dark mode" aesthetic, it features real-time search, user authentication, and a personalized watchlist system.

Key Features:

Cinematic Hero Interface: An immersive home page featuring dynamic video backgrounds and auto-playing trailers.

Real-Time Fuzzy Search: A high-speed search engine that filters through the movie catalog by title, genre, or year as you type.

Interactive Watchlist (My List): Users can save their favorite movies to a persistent "My List" stored in the browser's local storage.

Advanced Authentication: Custom Sign-In and Sign-Up modals with password strength indicators and real-time input validation.

Responsive Discovery: Categorized content rows (Trending, Action, Sci-Fi) with smooth-scrolling carousels optimized for all screen sizes.

User Profiles: A dedicated dashboard for users to manage their saved content and account details.
